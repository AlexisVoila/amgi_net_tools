# amgi_net_tools
___
Socks5 proxy server implementation

## Dependencies
___
* CMake
* C++ 17 
* C++ Boost
* OpenSSL

## Build instructions
#### Linux
    $ git clone https://github.com/alexisvoila/amgi_net_tools.git
    $ cd amgi_net_tools
    $ mkdir build
    $ cd build
    $ cmake ..
    $ make
